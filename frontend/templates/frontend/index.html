    ¿Aprobar esta nómina?
    Se registrará automáticamente una transacción de egreso en el ledger. Esta acción no se puede deshacer.
    ¿Copiar horario a la siguiente semana?
    Se crearán los mismos turnos para el empleado en la semana siguiente. Si ya existe un horario para esa semana, la operación fallará.
    ¿Marcar nómina como pagada?
    Indica que el pago físico fue realizado. El ledger ya fue actualizado al aprobar la nómina.
    ¿Marcar orden como recibida?
    El stock de cada producto se incrementará automáticamente. Esta acción no se puede deshacer.
    ¿Cancelar esta orden?
    La orden pasará a estado CANCELADA. El stock no se verá afectado.
    ¿Marcar solicitud como revisada?
    La solicitud cambiará a estado REVISADA. Puedes crear una orden de compra a partir de ella.
    ¿Cancelar esta venta?
    Esta acción restaurará el stock de todos los productos incluidos. No se puede deshacer.
